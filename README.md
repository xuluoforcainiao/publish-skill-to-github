# publish-skill-to-github

Publish or update QoderWork skills to GitHub as public repositories with versioned releases. Use when the user wants to upload skills to GitHub, sync skill changes to an existing repo, create releases, or when any skill modification needs to be mirrored to GitHub. Handles both first-time publishing and incremental updates.
